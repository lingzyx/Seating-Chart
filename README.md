# 禮堂座位安排工具

這是一個可以部署到 GitHub Pages 的純前端網頁應用程式。所有 Excel 讀取、賓客資料處理、座位拖拉、走道配置與 Excel 匯出都在瀏覽器端完成，不使用後端、不使用資料庫。

## 功能

- 上傳 `.xlsx` / `.xls` / `.csv` 賓客名單
- 支援欄位：機關單位、姓名、職稱、備註
- 支援只有一欄完整文字的名單，例如：`立法委員馬文君服務處王秘書小明`
- 自動辨識常見機關單位結尾詞：署、局、處、所、公司、工會、公會、中心等
- 左側未安排名單拖拉至右側座位
- 已安排賓客可重新拖拉；目標座位有人時會交換座位
- 可清除單一座位，讓賓客回到未安排名單
- 可自由設定排數、列數、垂直走道與水平走道
- 匯出 Excel 座位表，並自動產生「台下視角」與「台上視角」兩個工作表
- 匯出 Excel 中，每個座位一格、走道為空白欄/列、座位有框線、文字自動換行，座位格不設定底色
- 匯出 Excel 中，第一行機關單位為粗體，第二行姓名/職稱資訊為一般字體
- 「台上視角」會把座位表做 180 度旋轉，方便站在舞台上看台下時使用

## GitHub Pages 部署方式

1. 在 GitHub 建立一個新的 repository。
2. 把本專案所有檔案上傳到 repository 根目錄。
3. 到 repository 的 **Settings → Pages**。
4. Source 選擇 **Deploy from a branch**。
5. Branch 選擇 `main`，資料夾選擇 `/root`。
6. 儲存後等待 GitHub Pages 發布完成。

## Excel 名單格式範例

### 有欄位格式

| 機關單位 | 姓名 | 職稱 | 備註 |
| --- | --- | --- | --- |
| 南投縣政府 | 林小明 | 縣長 |  |
| 台電南投區處 | 王秀文 | 處長 |  |

匯出時會顯示：

```text
南投縣政府
林縣長小明
```

### 單欄完整文字格式

| 來賓 |
| --- |
| 立法委員馬文君服務處王秘書小明 |
| 台電南投區處林處長秀文 |

系統會盡量辨識成：

```text
立法委員馬文君服務處
王秘書小明
```

若辨識不正確，可在畫面中按「編輯」手動調整。

## 使用的前端套件

- SheetJS：瀏覽器端讀取 Excel
- ExcelJS：瀏覽器端產出 Excel，並支援同一儲存格內的局部粗體 rich text

這些套件透過 CDN 載入，因此不需要 npm install、不需要打包、不需要後端伺服器。
